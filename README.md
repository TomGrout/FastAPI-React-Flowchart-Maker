# fastAPI + React

## setup
Run in terminal:

cd frontend  
npm install  
npm run build  
npm start   

[open a new terminal instance]
cd backend  
python3 -m venv .venv
source .venv/bin/activate
pip install fastapi uvicorn
uvicorn main:app --reload  

= working app!  
-> frontend on port 3000
-> backend on port 8000

--- 

This app demonstrates React and fastAPI performance and code modularity
Use it to build flowcharts for LLM workflows and submit them to a backend store
